# DAP-S PIP plot with cluster / credible-set annotation

DAP-S PIP plot with cluster / credible-set annotation

## Usage

``` r
daps_plot(res, band = 0)
```

## Arguments

- res:

  A list of DAP-S fine-mapping results.

- band:

  Fraction of the row gap allowed for lifting between 0 and 1; default
  0.

## Value

A ggplot object (patchwork layout).
