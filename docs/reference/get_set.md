# DAP-S fine-mapping getting signal clusters or credible sets

DAP-S fine-mapping getting signal clusters or credible sets

## Usage

``` r
get_set(output, coverage)
```

## Arguments

- output:

  A list of DAP-S fine-mapping results.

- coverage:

  Desired coverage for credible sets. If coverage is NULL, return signal
  clusters instead of credible sets.

## Value

A list of signal clusters or credible sets.
